Anime & Manga Recommendations:

A modern, responsive web application that helps users discover and explore anime and manga based on preferences, genres, and popularity. Built with a focus on performance, user experience, and clean UI design.

Features:

* Search for anime and manga by title

*Filter content by genre, rating, and popularity

*Display detailed information (ratings, synopsis, episodes, etc.)

* Add items to favorites list

*Fully responsive design for mobile and desktop

* Fast and dynamic UI with real-time updates

 Tech Stack:

Frontend: HTML5, CSS3, JavaScript

Framework: React.js

State Management: React Hooks (useState, useEffect)

API: Jikan API (MyAnimeList unofficial API)

Styling: Custom CSS / Flexbox / Grid
